# CS-First-Year-Projects
A centralized repository for my technical explorations into Systems Architecture, Low-Level Logic, and C++ Programming. This project serves as a documentation of my academic progression at WSB University, focusing on building a "First Principles" understanding of computer science and cybersecurity.

Key Modules & Learnings
1. Control Flow & Decision Logic (Use_of_switch.cpp)
Objective: Implementing efficient multi-way branching.

Key Learning: Explored the performance benefits of switch statements over complex if-else chains in C++ and how they map to jump tables at the assembly level.

2. Functional Modularization (Creating_functions.cpp)
Objective: Enhancing code reusability and readability through functions.

Key Learning: Investigated function overloading rules and the mechanics of the call stack, ensuring clean separation of concerns within a program.

3. Iterative Structures (Intro_to_Loops.cpp)
Objective: Automating repetitive tasks through for and while loops.

Key Learning: Analyzed loop efficiency and exit conditions to prevent memory leaks and infinite execution—crucial for system stability and security.

4. Computational Logic (Arithmetic_calculations.cpp)
Objective: Performing data-driven mathematical operations.

Key Learning: Focused on binary representation and how C++ handles different data types (integers vs. floating-points) during arithmetic processing.

Tech Stack & Tools
Language: C++

Concepts: Binary Systems, Digital Logic, Control Flow, Cybersecurity Fundamentals.

Tools: Git, GitHub, VS Code.

Cybersecurity Connection
As a Cybersecurity student, I treat these fundamentals as the bedrock of secure coding. Understanding how a switch statement handles unexpected input or how a loop manages memory is the first step in preventing buffer overflows and other common vulnerabilities
